# Iconos de diferenciales

Sistema de cuatro iconos vectoriales para explicar los diferenciales centrales de AudioScript Contextual.

## Iconos

- `local-data-sovereignty.svg`: computadora local, carpeta de proyecto y escudo. Comunica que audio, transcripcion y memos viven en la maquina de quien investiga, sin subir voces a terceros.
- `methodological-control.svg`: documento de transcripcion, forma de onda, codigos y lapiz. Comunica control humano sobre la transcripcion automatica y puente entre producir texto y analizarlo.
- `accessible-cost.svg`: moneda, signo de infinito, usuarios y linea de cancelacion. Comunica ausencia de cobro por minuto y costo marginal cero despues de instalar.
- `offline-fieldwork.svg`: laptop, paisaje de campo, pin de ubicacion y Wi-Fi tachado. Comunica trabajo posible sin conexion confiable.

## Estilo

- Formato: SVG cuadrado `512 x 512`.
- Base visual: fondo marfil, azul profundo, verde de confianza, cobre para elementos activos y rojo solo para negaciones.
- Uso recomendado: diferenciales de producto, landing page, presentaciones, guia beta y materiales de onboarding.

Abre `icon-set-preview.html` para revisar el set completo en contexto.
